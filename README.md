### Installation

```shell
yarn install
```

### Development

```shell
yarn dev
```

### Build

```shell
yarn build
```

### Start built server

```shell
yarn start
```

### Testing

```shell
yarn test-dev
```

### Testing with watch

```shell
yarn test-dev
```

### Coverage

```shell
yarn coverage
```

### Coverage with watch

```shell
yarn coverage-dev
```

### Environments variables

| Name       | Default | Description                      |
| ---------- | ------- | -------------------------------- |
| PORT       | 8080    | Port of server                   |
| SECREY_KEY | secret  | Secret key used for jsonwebtoken |
